# MyApp
- Install NodeJs

> on PowerShell
`npm install -g @angular/cli`

`$env:NODE_OPTIONS = "--openssl-legacy-provider"`
`ng install`

`npm install -g firebase-tools`
`firebase login`
`firebase init`
